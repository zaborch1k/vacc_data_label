# Инструкция по установке / настройке

## Установка Label Studio
### 0. Проверьте, установлен ли питон
0.1. Откройте командную строку (cmd для винды, terminal для линукса)  
0.2. Скопируйте и вставьте в терминал:
```bash
pip3.12 --version
```
Если вы видите что-то вроде ```pip какие-то циферки ...```, нужный питон установлен. Переходите к [пункту 2](#2-установите-label-studio).  
Если вы видите что-то вроде ```не является внутренней или внешней командной``` или ```command not found```, нужный питон не установлен. Переходите к [пункту 1](#1-установите-питон).

### 1. Установите питон
1.1. Установите python3.12 (именно 12, а не 13; 13 не будет работать) из Microsoft Store (предпочтительно) или с официального сайта (https://www.python.org/downloads/release/python-3120/)  
1.2. Перейдите в [пункт 0](#0-проверьте-установлен-ли-питон) и повторите проверку

### 2. Установите Label Studio
2.1. Cкопируйте и вставьте в терминал (создает виртуальную оболочку питона):
```bash
python3.12 -m venv venv
```
#### Для винды:  
```bash
venv\Scripts\activate.bat
```
Слева от ```C:\Users\...``` должна появиться надпись ```(venv)```

#### Для не винды:
```bash
source venv/bin/activate
```
Слева от ```$``` должна появиться надпись ```(venv)```

2.2. Скопируйте и вставьте в терминал:
```bash
pip install label-studio
```
2.3. Ждите
2.4. Скопируйте и вставьте:
```bash
label-studio
```
Если вывод неадекватен (```command not found``` / ```не является внутренней или внешней командной``` / что-то красное), вы накосячили  
Если вывод типо ```=> Database and media.... много всяких красивых строк```, установка прошла успешно

## Настройка Label Studio
### 1. Запуск
1.1. Запустите в терминале:
```bash
label-studio
```
Не завершайте выполнение команды

1.2. Откройте http://localhost:8080/  
1.3. Зарегайтесь  

### 2. Настройка
Create project -> сверху:
1. Data Import -> Upload Files -> Выбираете часть файлов (все сразу выбрать не получится из-за лимита. Выберите 15-20 штук (условно))
2. Labeling Setup -> (слева, выбрана по дефолту) Computer Vision -> Object Detection with Bounding Boxes:  
    2.1. Удалите после Labels два примера Airplanes и Car, они не нужны  
    2.2. В Add new labels добавьте название своего label
3. Save
Вы создали проект, в котором будете работать
